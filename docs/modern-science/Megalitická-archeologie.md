# Megalitická archeologie (Megalithic Archaeology)

The study of megalithic structures worldwide — including [[Baalbek]], [[Stonehenge]], [[Tiwanaku]], and [[Gíza]].

## Key Mysteries

1. How were multi-ton stones moved and placed?
2. What technology was used for precise cutting?
3. What was the purpose of these structures?
4. Why do similar structures appear worldwide?

## Sitchin's View

Sitchin argued that these structures were built with Anunnaki technology — the stones cannot be explained by conventional ancient methods.
