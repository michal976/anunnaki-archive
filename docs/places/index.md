# Místa (Places)

This folder catalogs the key locations in the Anunnaki narrative as reconstructed by Zecharia Sitchin from Sumerian, Akkadian, Babylonian, and biblical sources. These sites span the geography of the ancient Near East and beyond, forming a network of settlements, space facilities, mining operations, and ceremonial centers.

## The Anunnaki Settlement Network

According to Sitchin, the Anunnaki established a structured network of settlements in the [E.Din](../concepts/E.DIN.md) (the Mesopotamian "Garden of Eden") and beyond:

| Location | Function |
|----------|----------|
| [Eridu](../concepts/Eridu.md) | First Anunnaki settlement on Earth |
| [Nippur](Nippur.md) | Mission Control / Central command |
| [Sippar](Sippar.md) | Spaceport / Landing site |
| [Baalbek](Baalbek.md) | Launch pad platform |
| [Giza](giza.md) | Navigation beacon (the Great Pyramid) |
| [Sinai](sinai.md) | Secondary spaceport |
| [Jerusalem](jerusalem.md) | "Mount of the Crossing" — guidance point |
| [Tilmun](Tilmun.md) | "Land of the Rocketships" |
| [Ararat](Ararat.md) | Landing point after the Flood |

## Mesopotamian City-States

The major cities of Sumer and Akkad each had a patron deity:

- [Ur](Ur.md) — Patron: [Nannar](../characters/Nannar.md) (Sin, the Moon God)
- [Uruk](Uruk.md) — Patron: [Inanna](../characters/Inanna.md) (Ishtar)
- [Babylon](Babylon.md) — Patron: [Marduk](../characters/Marduk.md)
- [Nippur](Nippur.md) — Patron: [Enlil](../characters/Enlil.md)
- [Eridu](../concepts/Eridu.md) — Patron: [Enki](../characters/Enki.md)
- [Sippar](Sippar.md) — Patron: [Utu](../characters/Utu.md) (Shamash)

## Beyond Mesopotamia

Sitchin also examined sites outside Mesopotamia:

- [Tiwanaku](Tiwanaku.md) — South American Anunnaki colony
- [Stonehenge](Stonehenge.md) — Astronomical observatory
- [Apsu](Apsu.md) — The primordial cosmic waters / abode of Enki
