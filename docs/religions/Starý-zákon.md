# Starý zákon (Old Testament)

Zecharia Sitchin argued that the [[Bible-Genesis|Old Testament]] (Hebrew Bible) is a monotheistic redaction of much older Sumerian texts about the [[Anunnaki]].

## Key Figures

| Biblical Figure | Sitchin's Identification |
|----------------|---------------------------|
| [[Yahweh]] / Elohim | [[Enlil]] (or the Anunnaki council) |
| [[El]] | [[Enlil]] |
| The "Sons of God" | The [[Anunnaki]] |
| [[Satan]] | Possibly [[Nergal]] or a tester god |
| [[Nephilim]] | Hybrid offspring of Anunnaki and humans |
| [[Moses]] | An Egyptian prince who knew the gods |
| [[Noah]] | [[Ziusudra]] / [[Atra-Hasis]] |
| [[Abraham]] | A chosen follower of Enlil/Yahweh |
