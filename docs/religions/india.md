# Indie (India — Vedic Religion)

Zecharia Sitchin explored connections between the Anunnaki narrative and the religious traditions of India, particularly the Vedas and Puranas.

## Vedic Parallels

| Vedic God | Possible Anunnaki Connection |
|-----------|------------------------------|
| Indra | Storm/weather controller — [Ishkur](../characters/Ishkur.md) |
| Agni | God of fire — [Gibil](../characters/Gibil.md) |
| Varuna | God of waters — [Enki](../characters/Enki.md) |
| Vishnu | The preserver / cosmic maintainer |
| Shiva | The destroyer — [Nergal](../characters/Nergal.md) |
| The '''Devas''' | The Anunnaki |
| The '''Asuras''' | A rival group of beings |

## Vimanas

Indian epics (the Ramayana, Mahabharata) describe flying machines ([Vimanas](../themes/vimanas-and-aircraft.md)) that Sitchin identified as Anunnaki spacecraft.
