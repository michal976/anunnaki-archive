# Starý zákon (Old Testament)

Zecharia Sitchin argued that the [Old Testament](../sources/Bible-Genesis.md) (Hebrew Bible) is a monotheistic redaction of much older Sumerian texts about the Anunnaki.

## Key Figures

| Biblical Figure | Sitchin's Identification |
|----------------|---------------------------|
| Yahweh / Elohim | [Enlil](../characters/Enlil.md) (or the Anunnaki council) |
| El | [Enlil](../characters/Enlil.md) |
| The "Sons of God" | The Anunnaki |
| Satan | Possibly [Nergal](../characters/Nergal.md) or a tester god |
| Nephilim | Hybrid offspring of Anunnaki and humans |
| Moses | An Egyptian prince who knew the gods |
| Noah | Ziusudra / [Atra Hasis](../sources/Atra-Hasis.md) |
| Abraham | A chosen follower of Enlil/Yahweh |
